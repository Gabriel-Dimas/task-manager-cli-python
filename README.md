# 📝 Task Manager CLI (Python) <img src="https://cdn.jsdelivr.net/gh/devicons/devicon@latest/icons/python/python-original.svg" width="30" />

Um gerenciador de tarefas simples via terminal (CLI), desenvolvido em Python.  
Permite adicionar, listar, atualizar, concluir e remover tarefas, com persistência em arquivo JSON.

---

## 🚀 Funcionalidades

- ➕ Adicionar tarefas  
- 📋 Listar tarefas  
- ✏️ Atualizar tarefas  
- ✔️ Marcar tarefas como concluídas  
- ❌ Deletar tarefas  
- 💾 Salvamento automático em arquivo JSON  

---

## 🛠️ Tecnologias utilizadas

- Python 3  
- JSON (para armazenamento de dados)  
- Terminal / CLI  

---

## 📂 Estrutura do projeto

├── main.py

├── tarefas.json

---


## ▶️ Como executar

1. Clone o repositório:
2. Acesse a pasta: cd task-manager-cli-python
3. Execute o programa: python main.py
